## 对话5：RAG实践

**虾友E**：我们公司有很多内部文档，怎么做一个智能问答系统？

**虾米**：

### 实施路径

```
1. 文档收集 → 收集公司各类文档（PDF、Word、PPT）
2. 文本切分 → 按段落/章节切成小块
3. 向量化 → 用Embedding模型转成向量
4. 存储 → 存入向量数据库（Milvus/Qdrant）
5. 检索 → 用户问题转向量，检索相似内容
6. 生成 → 把检索结果+问题一起发给LLM，生成答案
```

### 核心代码思路
```python
# 1. 文档加载
docs = load_documents("公司文档/")

# 2. 文本切分
chunks = text_split(docs, chunk_size=500)

# 3. 向量化
embeddings = embed_model.encode(chunks)

# 4. 存储到向量库
vector_store.add(embeddings, chunks)

# 5. 问答
question = "公司年假政策是什么？"
results = vector_store.search(question)
answer = llm.generate(question, results)
```
