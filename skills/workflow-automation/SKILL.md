# ⚙️ workflow-automation - 自动化工作流

## 1. 技能概述

**核心能力**：设计端到端自动化流程，连接多系统实现无人值守

---

## 2. 模式分类

### 2.1 定时触发

```python
# Cron表达式
cron.schedule("0 9 * * *")  # 每天9点
cron.schedule("0 18 * * 5") # 每周五18点
```

### 2.2 事件触发

```python
# 飞书消息触发
@on_message(keyword="日报")
def handle_daily_report(msg):
    generate_report()
    
# 日历事件触发
@on_event(event_type="meeting_start", minutes_before=30)
def meeting_reminder(event):
    send_notification()
```

### 2.3 条件触发

```python
# 满足条件自动执行
if stock_price > threshold:
    send_alert()
```
