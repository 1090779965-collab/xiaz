# 推荐适用Skill清单

## 1. Task Decomposition & Execution Skill
**用途**：将复杂任务拆解为可执行的子任务，结合CoT（思维链）和ReACT（推理-行动循环）逐步执行。
**适用场景**：规划项目、解决多步骤问题、不确定执行路径时。
**核心技术**：子任务分解、步骤检查、动态调整。

## 2. Model Distillation Skill
**用途**：将大模型压缩为小模型，保留性能同时降低资源消耗。
**适用场景**：边缘设备部署、降低推理成本、加速模型响应。
**核心技术**：温度参数调整、KL散度损失、知识迁移。

## 3. LoRA Fine-tuning Skill
**用途**：高效微调模型，无需全量训练。
**适用场景**：领域适配、小样本学习、快速模型迭代。
**核心技术**：低秩矩阵、缩放系数α、参数冻结。

## 4. Agent Memory with RAG Skill
**用途**：构建具备长期记忆的智能Agent。
**适用场景**：聊天机器人、个性化助手、知识密集型任务。
**核心技术**：分层记忆存储、向量数据库检索、动态知识更新。

## 5. Context Extension with RoPE Skill
**用途**：处理超长文本，突破模型上下文限制。
**适用场景**：长文档分析、法律合同审查、代码库理解。
**核心技术**：旋转位置编码、NTK插值、RingAttention。

## 6. Domain Fine-tuning Skill
**用途**：将通用模型转化为领域专家。
**适用场景**：医疗诊断、金融分析、法律文书处理。
**核心技术**：数据构建、分阶段训练、偏好对齐。

---
**优先级建议**：
1. **Task Decomposition**（基础能力）
2. **LoRA Fine-tuning**（高效模型适配）
3. **Agent Memory with RAG**（智能Agent核心）

这些Skill均基于今日学习内容，可直接落地应用。
