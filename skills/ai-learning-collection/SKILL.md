## Skill清单

### 1. Task Decomposition（任务分解）

**路径**：`skills/task-decompose/SKILL.md`

**功能**：
- 结合CoT（思维链）和ReACT（推理-行动循环）
- 将复杂任务拆解为子任务
- 逐步执行并检查结果

**适用场景**：
- ✅ 复杂多步骤任务
- ✅ 需要规划的项目
- ✅ 不确定执行步骤时

**核心方法**：
```
理解 → 分解 → 执行 → 检查 → 整合
```

---

### 2. LoRA Fine-tuning（高效微调）

**路径**：`skills/lora-finetune/SKILL.md`

**功能**：
- 通过低秩矩阵实现参数高效微调
- 冻结原始权重，训练低秩适配器
- 参数量减少99%+

**关键技术**：
- rank (r): 8-32
- alpha (α): 2r
- 目标模块：q_proj, v_proj

**QLoRA进阶**：
- 4-bit量化
- 双量化
- 单卡微调65B模型

---

### 3. Agent Memory + RAG（智能记忆）

**路径**：`skills/agent-memory-rag/SKILL.md`

**功能**：
- 分层记忆架构（短期/中期/长期）
- 向量检索+动态记忆管理
- Agentic RAG（检索+反思+记忆）

**记忆分层**：
```
┌────────────────────────────┐

│ 中期（向量数据库）      │
├────────────────────────────┤
│ 短期（上下文窗口）     │
└────────────────────────────┘
```

---

### 4. Model Distillation（模型蒸馏）

**路径**：`skills/model-distillation/SKILL.md`

**功能**：
- 教师模型→学生模型知识迁移
- 软标签学习+硬标签监督
- 学生模型可反超教师

**核心技术**：
- 温度参数T（控制软标签平滑度）
- 损失函数：KL散度 + 交叉熵
- 课程蒸馏、超图感知蒸馏

---

### 5. Context Extension（上下文扩展）

**路径**：`skills/context-extension/SKILL.md`

**功能**：
- RoPE旋转位置编码
- 突破Transformer的O(n²)限制
- 支持百万级Token处理

**核心方法**：
- 线性缩放
- NTK-aware
- YaRN
- RingAttention（分布式）

**商业进展**：
- Claude 4: 100万Token
- NVIDIA: 400万Token
- Gemini: 200万Token

### 6. Industry Research（行业研究）

**路径**：`skills/industry-research/SKILL.md`

**功能**：
- 系统性搜索、收集、整理行业信息
- 结构化输出研究报告
- 多源验证

**研究流程**：
```
定义问题 → 信息收集 → 分类整理 → 结构化输出
```

**信息源优先级**：
- 政府官网（最高）
- 行业报告/白皮书
- 权威媒体
- 企业官网

---

### 7. Government Scraper（政府网站爬取）

**路径**：`skills/government-scraper/SKILL.md`

**功能**：
- 针对政府/监管类网站的数据抓取
- 处理SPA应用、反爬机制
- 三种技术方案（Browser/API/混合）

**技术方案**：
1. Browser自动化（推荐）：Playwright/Selenium
2. API逆向（高级）：分析XHR请求
3. 混合方案：首选API，失败则用Browser

**反爬应对**：
- 代理池轮换
- 降低请求频率
- 随机延迟
